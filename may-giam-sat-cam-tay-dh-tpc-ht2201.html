Chi tiết kỹ thuật

System Overview

Dahua thermal cameras characterized by non-contact, multi-objective, rapid, and no risk to temperature testers, etc. It can be conveniently deployed in public places with dense personnel, and can be used for efficient and safe preliminary screening of temperature anomalies.

Thermal

  • Detector Type : Uncooled VOx
  • Effective Pixels : 256 × 192
  • Pixel Pitch : 12 μm
  • Spectral Range : 8 μm–14 μm
  • Thermal Sensitivity (NETD) : ≤50 mK@f/1.0
  • Focal Length : 3.5 mm
  • Field of View : 37.8° × 50.6°
  • Focus Mode : Athermalized
  • Aperture : F1.0
  • Color Palettes : 4(Whitehot/Blackhot/Rainbow//Ironbow)

Image Display

  • Display : 2.4′ LCD
  • Resolution : 320 × 240
  • Temperature Measurement
  • Temperature Measurement Range : 30°C ~ 45°C (95°F-113°F)
  • Temperature Accuracy : ±0.5°C (Operating Temperature 15°C ~35°C )
  • Temperature Mode : Center/Max. Temperature
  • Temperature Measuring Distance : Min.0.5m

Storage

  • Image storage : Micro SD Card (Max 256G)

Port

  • USB : Micro USB
  • Tripod screw hole : Yes

Battery

  • The battery type : Built-in rechargeable lithium-ion battery
  • Battery Life : >8h continuous operation @ 25°C
  • Charging time : ≤2.5h@ 25°C

Environment

  • Operating Temperature : 15°C to 35°C (59°F-95°F)
  • Storage Temperature : -20°C to +60°C (-4°F-+140°F)

Physical Characteristics

  • Protection Grade : IP54
  • Packaging Dimensions : 192.4mm×62.55mm×72.2mm
  • Net Weight : <500g

Note: The temperature measurement accuracy is the typical value under the specified mode and application conditions, and the final interpretation belongs to Dahua
